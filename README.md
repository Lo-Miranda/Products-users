# Products-users
Projeto realizado seguindo o curso Udemy Nélio Alves - Java Completo.
A estrutura mostra a relação entre usuários, pedidos, itens de pedidos, categorias de pedidos e pagamento.

Conceitos presentes:

- Spring Boot Java;
- Maven;
- Postman;
- Modelo de domínio;
- Rest API;
- CRUD;
- Tratamento de exceções
